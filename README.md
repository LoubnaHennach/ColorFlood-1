## Instructions d'utilisation

Grâce au Makefile,
* pour compiler

```
make
```
* pour tester l'absence de fuites de mémoire (Valgrind)

```
make valgrind
```
* pour les tests unitaires

```
make ??
```
* pour générer la documentation (Doxygen)

```
make doxygen
```

* pour nettoyer le répertoire
```
make clean
```
